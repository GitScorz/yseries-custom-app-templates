# YFlip Phone - Custom App Templates

### The repository contains templates for creating custom apps for the [YFlip Phone](https://www.yflip-phone.com/).

## Templates provided

- React + Typescript
- React + Javascript (soon)
- Vanilla Js (soon)

- Could add more on demand.
